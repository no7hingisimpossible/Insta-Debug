How does javascript work?

Javascript runs on the browser with the help of V8 engine and it is an interpreted language which uses JIT also known as Just In Time comipler side by side to execute the codes faster. It is a higher levellanguage so it is unreadable to the machine. JIT translates the codes into machine readable codes also known as byte codes after which the codes are executed.

Local Storage Vs Session Storage

Unlike cookies, local and session storages cannot be accessed by the servers which makes it secure. Data is saved locally in both local storage and session storage.
Local storage stores datas without any time limitations and the datas will not be deleted even after the browser tab is closed.
Session storage stores datas just like as local storage but the difference is that the data stored will be immediately lost or deleted as soon as the browser tab is closed.
